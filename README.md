### 🕸️ ACTIVE PROJECT NODES (FEATURED PROJECTS)

#### ⚡ [Synapse Notes Web Application](https://github.com/Wired-Navi0x1F/synapse-notes)
*A dynamic markdown-based knowledge management web app featuring 3D node-link relationship mapping.*
*   **Key Implementations:** Engineered guest authentication, secure archive vaulting, markdown rendering, and "The Wired" (an interactive **Three.js** 3D node network visualizer graphing connections between notes).
*   **Administration:** Developed an admin dashboard supporting activity logging, moderation workflows, and database backup management.
*   **Tech Stack:** `Flask`, `MySQL`, `JavaScript`, `Three.js`, `Python`
*   **Links:** [💻 Code Repository](https://github.com/Wired-Navi0x1F/synapse-notes)

#### 🚘 [Bayesian MPC Predictive Agent](https://github.com/Wired-Navi0x1F/Enigmaa)
*Trajectory prediction and active safety-shield agent for autonomous vehicles.*
*   **Key Implementations:** Developed a GMM-based trajectory prediction neural model and Monte Carlo dropout for uncertainty estimation. Built Hard Shield AEB (Automatic Emergency Braking) modules for collision prevention at intersections and two-way roads.
*   **Simulation & Explainability:** Deployed **ROS2** communication nodes for vehicle simulation workflows and applied Integrated Gradients techniques for decision explainability.
*   **Tech Stack:** `PyTorch`, `highway-env`, `Gymnasium`, `ROS2`, `Python`
*   **Links:** [💻 Code Repository](https://github.com/Wired-Navi0x1F/Enigmaa)
